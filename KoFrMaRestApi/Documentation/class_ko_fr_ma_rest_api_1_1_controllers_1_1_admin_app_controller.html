Detailed Description

Used for communication between AdminApp and RestApi AUTHENTICATION: Use valid token in PostAdmin.adminInfo retrievable from RegisterToken(AdminLogin) Http codes:

  • 400 Bad Request - invalid input
  • 401 Unauthorized - invalid token or username
  • 403 Forbidden - insufficient permission
  • 200 OK - OK
  • 500 Internal Server Error - Something when wrong on the server, check ServerExceptions table in AdminApp to see more details

Permission codes: 1 - Add Admins 2 - Add Tasks 3 - Change Table Data 4 - Change Permissions 5 - Change Passwords
